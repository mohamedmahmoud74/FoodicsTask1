Google Search Test
This project is a Java-based automated test script that uses Selenium WebDriver 

Prerequisites
Java Development Kit (JDK) version 8 or later
Apache Maven build automation tool
Google Chrome web browser

Running the Test
To run the test, follow these steps:

Run the following command to run the test script: mvn test
The test script will open a Google search page, perform a search for the keywords "Mohammed Salah" and "Mohamed Abo Treka", and validate the results.


Additional Notes
Please ensure that you have a stable internet connection before running the test script.
